# ecommerce2 - Simple E-commerce (ESM)

## Quick start
1. Install dependencies in backend folder:
   ```bash
   cd backend
   npm init -y
   npm install express mongoose cors bcryptjs jsonwebtoken dotenv
   ```
2. Run MongoDB locally or use Atlas.
3. Start server:
   ```bash
   node app.js
   ```
4. Open `frontend/home.html` and `frontend/admin.html` in browser.
